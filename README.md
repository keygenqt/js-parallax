## Usage

```js
$('#block1').parallax({
    image: '/images/common/block1.jpg'
});

$('#block2').parallax({
    image: '/images/common/block2.jpg'
});

$('#block3').parallax({
    image: '/images/common/block3.jpg'
});
```
